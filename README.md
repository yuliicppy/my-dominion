## 概要

デッキ構築型ボードゲーム「ドミニオン」の基本セット（第2版）を、ブラウザ上でプレイできるようにするプロジェクトです。
まずはソロプレイ（プレイヤー1人）を想定し、自分一人でターンを進めてゲーム終了まで遊べる状態を目指します。

## 目的

TypeScript と React を用いたフロントエンド開発の理解を深めることを目的としています。
状態管理やUI構築、ゲームロジックの実装を通じて、React の実践的なスキル習得を目指します。

## 技術スタック

TypeScript

React

（必要に応じて今後追加：Vite / Next.js / Zustand / Redux Toolkit など）

## 機能要件
- ドミニオン基本セット（第2版）のカードを実装
- ソロプレイ可能（プレイヤーは1人想定）
- 手番進行（アクションフェーズ → 購入フェーズ → クリーンアップ）
- デッキ・捨て札・手札の管理
- カード効果の適用処理
- ゲーム終了条件の判定（属州枯渇など）
